# Laundry Services — Assignment

This is a small static web app scaffold for the Laundry Services project assignment. It implements a responsive UI with a booking form, services list, and a newsletter subscription (demo-only — stores data in `localStorage`).

How to view

- Open `d:/Assignment4/index.html` in your browser.

What I implemented

- `index.html` — main UI and booking modal.
- `css/styles.css` — layout and responsive styles.
- `js/app.js` — client-side form handling and demo persistence.
